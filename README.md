# Kering Acquisition of Brunello Cucinelli – M&A & LBO Model

## Overview

This project analyses a hypothetical acquisition of Brunello Cucinelli S.p.A. by Kering S.A.

The objective is to evaluate transaction economics, strategic rationale, shareholder impact and private equity returns through a fully integrated Excel model.

## Key Components

### M&A Analysis
- Enterprise Value and Equity Value calculations
- Acquisition premium analysis
- Revenue and cost synergy assumptions
- Pro Forma financial statements
- EPS Accretion / Dilution analysis
- Transaction impact assessment

### LBO Analysis
- Sources & Uses framework
- Debt financing assumptions
- Sponsor equity contribution
- MOIC calculation
- IRR analysis
- Exit multiple sensitivity

## Strategic Rationale

The model explores whether Brunello Cucinelli's ultra-luxury positioning could complement Kering's portfolio and create shareholder value through operational and commercial synergies.
